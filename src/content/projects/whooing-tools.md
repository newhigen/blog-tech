## 개요

복식부기 가계부 서비스인 후잉을 더 편하게 쓰기 위해 만든 스크립트와 문서 모음이다. 핵심은 은행 원본 데이터를 후잉 입력 형식으로 바꾸고, 월별 정산 규칙을 일관되게 유지하는 것이다.

내부적으로는 `Whooing Tools`라고 부르고 있다.

## 주요 기능

- **명세서 변환** — 은행 거래내역을 후잉 import CSV로 변환
- **워크플로 문서화** — 계정 구조와 월간 정산 규칙 정리
- **리뷰 기록** — 월별 거래 검토 메모 누적

## 구현 포인트

- Python 스크립트로 반복 수작업 제거
- 데이터 정제 규칙과 회계 규칙을 같이 문서화
- 단순 스크립트보다 운영 가능한 개인 시스템에 가깝게 구성

## 만든 이유

가계부는 입력보다 **지속 가능한 정리 규칙**이 더 중요하다. 수작업 변환과 매번 달라지는 분류 기준을 줄이기 위해 도구와 문서를 함께 만들었다.
