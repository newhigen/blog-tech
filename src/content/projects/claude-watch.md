## 소개

Claude Code를 여러 개 동시에 켜놓고 작업하다 보면, 어느 창이 내 입력을 기다리고 있는지 일일이 확인하기 번거롭다. 모든 세션의 상태를 한 화면에 모아 "지금 대기 중인 창"을 바로 찾게 해준다.

## 주요 기능

- **세션 그리드** — 실행 중인 Claude Code 창을 한눈에
- **대기 표시** — 내 답을 기다리는 창 강조
- **상태 모니터링** — 작업 중·완료·대기 구분

## 배경

병렬로 여러 에이전트를 돌릴수록 "다음에 손볼 창"을 빠르게 찾는 게 생산성을 좌우한다.
