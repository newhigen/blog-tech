## 개요

가입한 보험과 제안받은 보험을 같은 기준으로 비교하기 위해 만든 개인용 분석 도구다. 상품 설명서와 PDF를 그대로 읽는 대신 **보험료, 보장 범위, 조합 효과**를 빠르게 확인할 수 있게 구성했다.

내부 파일명은 `Insurance Dashboard`다.

## 주요 기능

- **상품 비교** — 보험료와 핵심 담보를 같은 축에서 비교
- **실손 구조 요약** — 세대별 차이와 갱신형 특성 정리
- **정액 보장 조합 보기** — 중복 수령 가능성 확인
- **의사결정 보조** — A안/B안과 공통 조건을 분리해 정리

## 구현 포인트

- 브라우저에서 바로 여는 정적 HTML 대시보드
- 한국 보험 문맥에 맞는 설명과 표 구성
- 단순 시각화보다 실제 비교 판단에 필요한 텍스트 구조 강조

## 만든 이유

보험은 문서가 길고 용어가 복잡해서 비교가 어렵다. 특히 실손과 정액형이 섞이면 “무엇이 중복되고 무엇이 아닌지”가 흐려지기 때문에, 그 차이를 명확하게 보이게 하고 싶었다.
