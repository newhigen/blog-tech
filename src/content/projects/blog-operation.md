## 소개

글을 쓰고 싶었다. 그런데 플랫폼에 맡기면 언제든 사라질 수 있고, 디자인을 내 마음대로 바꾸기 어렵다. 그래서 직접 만들었다.

두 블로그는 성격이 다르다. **글쓰기 블로그**는 읽은 것, 생각한 것을 기록하는 공간이고, **기술 블로그**는 개발 환경, 도구, ML에 대한 글을 모아두는 공간이다.

## 주요 기능

- **독서 히트맵** — 연도별·월별 독서 기록을 한눈에 확인
- **포스트 분류** — 글쓰기·독서·생각 카테고리 필터링
- **다크/라이트 모드** — 시스템 설정 연동 + 수동 토글
- **TOC·사이드바** — 긴 글 탐색을 위한 자동 목차

## 운영 방식

Obsidian에서 작성하고 GitHub에 올리면 Netlify가 자동 배포한다. 코드 수정은 Claude Code로 처리한다. 직접 만든 블로그라 UI는 계속 조금씩 바뀐다.
