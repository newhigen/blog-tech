## 소개

Claude Code는 자주 업데이트된다. 매 릴리스 changelog를 다 읽기는 번거롭고, 정작 "지금 나에게 쓸모 있는 변화"는 묻히기 쉽다. 그래서 새 버전마다 핵심만 골라 정리하는 트래킹 페이지를 만들었다.

## 주요 기능

- **릴리스별 분류** — 기능·수정·실험으로 변경사항 정리
- **Pick** — 당장 써볼 만한 것 큐레이션
- **카테고리 뷰** — 주제별로 모아보기

## 배경

변화가 빠른 도구일수록 전체를 따라가기보다 "지금 쓸모 있는 변화"를 빠르게 집어내는 게 중요하다.
