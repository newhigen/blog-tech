## 개요

시프티에서 추출한 근태 데이터를 읽어 연간 히트맵, 출퇴근 분포, 월별 패턴, 특이 일정을 한 화면에서 보게 만든 대시보드다.

내부적으로는 `Shiftee Dashboard`라고 부르고 있다.

## 주요 기능

- **연간 히트맵** — GitHub 스타일 셀로 근무 유형과 야근 빈도 표시
- **출근/퇴근 분포** — 시간대별 히스토그램
- **월별 타임라인** — 연도별 평균 출퇴근 흐름 비교
- **특이 일정 요약** — 출장, 군사훈련, 휴가를 별도로 묶어 표시

## 구현 포인트

- 단일 `index.html`로 동작
- `days.json` 기반 데이터 로드
- Chart.js와 annotation 플러그인 사용
- Toss 계열의 조용한 다크 UI로 정리

## 만든 이유

엑셀 원본으로는 “언제 출근이 늦어졌는지”, “야근이 어느 시기에 몰렸는지”, “출장이 패턴에 어떤 영향을 줬는지”를 보기 어려웠다. 시계열과 분포를 같이 보는 대시보드가 필요했다.
