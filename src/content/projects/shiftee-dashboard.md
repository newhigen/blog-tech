## 문제 정의

엑셀 원본 근태 데이터로는 패턴이 보이지 않는다. "언제 출근이 늦어졌는지", "야근이 어느 시기에 몰렸는지", "출장이 리듬에 어떤 영향을 줬는지"를 보려면 시계열과 분포를 함께 봐야 한다.

시프티 앱에서 내보낸 데이터는 Excel 형식이라 그대로는 시각화가 안 된다.

## 과정

1. **데이터 추출** — 시프티 앱에서 근태 기록 Excel 내보내기.
2. **전처리** — `analyze.py`로 날짜·시각·근무 유형을 정규화해 `days.json`으로 변환. 야근 판별 기준(`ef` 기준, 퇴근 시각 아님)도 이 단계에서 적용.
3. **대시보드 설계** — 어떤 차트 조합이 패턴을 잘 드러내는지 여러 형태를 실험. `ot_compare.html`로 야근 표시 방식 비교.
4. **구현** — Chart.js 4.4.4와 annotation 플러그인으로 단일 `index.html` 구성. Toss 다크모드 디자인 시스템(`DESIGN.md`) 기반으로 UI 정리.

## 주요 기능

- **연간 히트맵** — 2023~2026년 GitHub 스타일 셀 그리드. 근무 유형(출장/연차/군사훈련)과 야근 빈도를 색상으로 구분
- **출근·퇴근 분포** — 시간대별 히스토그램으로 출퇴근 패턴 확인
- **월별 타임라인** — 연도별 평균 출퇴근 흐름을 오버레이로 비교
- **특이 일정 요약** — 출장·군사훈련·휴가를 그룹별로 분리해서 표시

## 구현 포인트

- 단일 `index.html` — 외부 서버 없이 브라우저에서 바로 실행
- `days.json` 기반 데이터 로드 (`ef` 기준 야근 재계산)
- Toss 계열의 조용한 다크 UI로 시각적 일관성 유지
